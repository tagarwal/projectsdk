Welcome to Java for Cloud (J4C)! J4C is a collection of Java libraries for writing cloud native applications. The core libraries of J4C are:

  • Rx Web Server: a highly performant reactive style HTTP core that provides the backbone for your REST service.

  • Config: a simple yet flexible configuration framework used by other J4C components

  • Security: a robust security provider that supports…​

  • JaNA (Java NoSQL API): a NoSQL API that enables you to write portable code, independent of specific NoSQL databases. JaNA supports multiple NoSQL databases such as MongoDB, DynamoDB and Cassandra.

  • JADE (Java API for Distributed Eventing): a Messaging API that enables you to write portable code, independent of specific messaging providers. JADE supports multiple messaging implementations such as Kafka and Kinesis

  • Resiliency: a client side liberary that provides multiple types of fault tolerance controls.

J4C also supports popular Java EE APIs such as JAX-RS (Jersey), JSON-P/B, JPA, CDI (Weld), and more.

You can run J4C applications locally on your system, or package them into docker containers. J4C also provides support for deploying your applications to Kubernetes.

To learn more about J4C checkout our Developer Guide. Or if you want to get started then dive right into our Getting Started Guide.
