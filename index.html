Java for Cloud

  More Libraries Less Framework

J4C is not a new framework to learn. Instead it is a currated list of libraries combined with Kubernetes integration to let you easily write and deploy your Java cloud native application.

  Lightweight Java SE Runtime

Your applicaion is just a Java SE program. No heavyweight containers here. The J4C Rx Web Server library supports Netty for HTTP support, as well as Jersey for more advanced REST development.

  Some New and Some Old Favorites

J4C libraries include some that are brand new such as Rx Web Server, Config and Cloud security. Others are old friends from Java EE: Jersey, Weld, JSON-B/P. And still others are popular open source libraries.

  Familiar Tools

No SDK. Just use the tools you already use: Git, Maven and Docker.

  Kubernetes Integration

Build and run your application locally, and then easily deploy to Kubernetes.

  Get Started Today

Quickly generate a J4C project using our Project Starter, or clone our sample repository and give our examples a try.
